Module Information and Specifications

Description

The T-Connect-Pro is based on the ESP32S3 main control chip and consists of a three-layer stacked board design. It offers a wide range of features and integrates three different communication modules‚ÄîCAN, RS485, and RS232‚Äîfor long-distance transmission. Additionally, it includes an Ethernet interface, a relay interface, an LoRa module (SX1262), and an LCD screen for more convenient operation.
